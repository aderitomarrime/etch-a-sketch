## Overview
A website version of an Etch-a-Sketch that changes the background colour of each square on every interaction.


## Live Demo

Visit the website: [Live Demo](https://aderitomaraderitomarrime/etch-a-sketch/)


## Features

- Minimal interface 
- On each square, mouse hovering changes the background to a random colour
- On each square, mouse hovering increases the opacity by 10%
- Allow the user to change the number of squares, between 1–100


## Technologies Used

- HTML5  
- CSS3 
- JavaScript


## Installation

To run this project locally, follow these steps:

1. Clone the repository:

```sh
git clone https://github.com/aderitomarrime/etch-a-sketch.git
```

2. Navigate to the project 
```sh
cd etch-a-sketch
```
3. Open index.html in your browser


## How to use

Once the website is open, move your mouse over any of the squares to change its background colour. You can also change the number of squares by clicking the “Change Grid Size” button, entering the desired number of squares, and pressing Enter.
